## Dataset 

**Nome:** Emergency Service – Triage Application  
**Link:** Kaggle – [https://www.kaggle.com/datasets/owjilab/emergency-service-triage-application](https://www.kaggle.com/datasets/owjilab/emergency-service-triage-application)  
**Descrição:** Conjunto de dados clínicos de atendimentos emergenciais contendo informações demográficas, sinais vitais e classificações de triagem realizadas por especialistas.  
**Principais colunas:**
- `Age`: Idade do paciente  
- `Sex`: Sexo biológico  
- `SBP` e `DBP`: Pressões sistólica e diastólica  
- `HR`: Frequência cardíaca  
- `RR`: Frequência respiratória  
- `BT`: Temperatura corporal  
- `NRS_pain`: Escala de dor  
- `Chief_complain`: Queixa principal do paciente  
- `KTAS_expert`: Nível de triagem atribuído por especialista  

O dataset foi utilizado para treinar e validar o modelo preditivo de triagem hospitalar (XGBoost balanceado, versão V4).
